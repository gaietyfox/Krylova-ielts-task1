# Krylova-ielts-task1
# Анализ корпуса текстов IELTS Writing Task 1


## Цель проекта

Создание корпуса текстов IELTS Writing Task 1 и разработка инструмента для автоматизированного анализа письменной речи.

---

## Описание проекта

Данный проект посвящён созданию и анализу корпуса текстов IELTS Writing Task 1.  
Работа включает этапы предобработки, частотного и морфологического анализа, а также расчёт показателей лексического разнообразия.

Проект состоит из двух логических частей:

1. Формирование и анализ корпуса
2. Реализация Telegram-бота для автоматической обработки текстов

---

## 1. Формирование и анализ корпуса

На первом этапе был сформирован корпус текстов IELTS Writing Task 1.

В рамках обработки были реализованы:

- очистка текстов
- токенизация
- удаление служебных элементов
- подсчёт длины текстов
- частотный анализ
- анализ глагольных форм
- выделение пар и вариантов конструкций
- расчёт показателя TTR (Type-Token Ratio)

Результатом являются структурированные данные в формате JSON.

---

## 2. Telegram-бот

На основе подготовленного корпуса реализован Telegram-бот, который:

- принимает текст пользователя
- анализирует длину и структуру текста
- использует подготовленные лингвистические ресурсы из анализа корпуса

Бот реализует базовый механизм автоматизированной лингвистической обработки текста.

## ЗАПУСК ПРОЕКТА

### 1. Установка зависимостей

Перед запуском необходимо установить используемые библиотеки:

```pip install pandas numpy matplotlib nltk wordcloud breame aiogram```

После установки (при работе в Jupyter) может потребоваться перезапуск ядра.

---

### 2. Запуск анализа корпуса

Откройте файл:

```Krylova_project_IELTS_WritingTask1.ipynb```

и выполните все ячейки

В результате будут:

- загружены и очищены тексты корпуса
- рассчитаны частотные и лингвистические показатели
- сформированы JSON-файлы:
  - pairs.json (коллокации verb + adverb)
  - lengths.json (длина текстов и тип задания)

---

### 3. Запуск Telegram-бота

После генерации JSON-файлов выполните в терминале:

```python bot.py```

Так как бот пока что локальный, перед запуском необходимо:

- создать Telegram-бота через @BotFather
- вставить полученный токен в файл bot.py
- убедиться, что файл pairs.json и lengths.json находятся в той же папке

Бот использует данные, полученные на этапе корпусного анализа.

---

### 4. Запуск в Google Colab (опционально)

Если проект запускается в Colab:

1. Склонируйте репозиторий:
   ```!git clone https://github.com/gaietyfox/Krylova-ielts-task1.git```

2. Перейдите в папку проекта:
   ```%cd /content/Krylova-ielts-task1```
